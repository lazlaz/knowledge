# Java 二进制和位运算

Java语言支持的位运算符还是非常多的，列出如下：  
* &：按位与
* |：按位或
* ~：按位非
* ^：按位异或
* <<：左位移运算符
* >>：右位移运算符
* >>>：无符号右移运算符


### &：按位与  
操作规则：同为1则1，否则为0。仅当两个操作数都为1时，输出结果才为1，否则为0。

### ~：按位非
操作规则：0为1，1为0。全部的0置为1，1置为0。

### ^：按位异或
操作规则：相同为0，不同为1。操作数不同时（1遇上0，0遇上1）对应的输出结果才为1，否则为0。

### <<：按位左移
操作规则：把一个数的全部位数都向左移动若干位。

### >>：按位右移
操作规则：把一个数的全部位数都向右移动若干位。
